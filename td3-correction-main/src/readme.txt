Cette application permet de recherche un pokemon par son nom
(complet ou partiel) et affiche les correspondances.

Nécessite une version récente de Java ;-).

Exemple:

$ java CatchEmAll chu 
Search results for « chu » among 800 pokemons : 
* Pikachu (35)  -  Electric
* Raichu (60)  -  Electric
* Pichu (20)  -  Electric
* Smoochum (45)  -  Ice
